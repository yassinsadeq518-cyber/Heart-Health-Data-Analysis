# Heart-Health-Data-Analysis
# 🩺 Heart Health Insights | Clinical Data Dashboard

## 📌 Project Overview
A healthcare analytics project designed to explore clinical datasets and identify key biometrics and risk factors highly correlated with heart disease and cardiovascular injuries.

## 📊 Key Insights & Features
* **Demographic Breakdown:** Visualizes heart injury distribution filtered by age groups and gender profiles.
* **Clinical Correlations:** Analyzes relationships between critical medical parameters such as blood pressure (`trestbps`), cholesterol (`chol`), and chest pain types (`cp`) against patient clinical outcomes.
* **Interactive Diagnostic Filters:** Dynamic slicers allowing deep dives based on resting ECG results and fasting blood sugar levels.

## 🛠️ Tech Stack & Skills
* **Tool:** Power BI Desktop
* **Healthcare Analytics:** Interpreting medical terminology into structured analytical dimensions.
* **Data Exploration:** Correlation mapping using advanced distributions.
